Add Bootstrap header to My Space

Posted on 2020-01-09 | In Building My Space

My Space is a community style website.The target of creating this website is to learn Spring Boot.
The first step is to add a header. Here I am using Bootstrap as the format.

  1. Download Bootstrap and import to IDE (IntellJ Idea).

    • Unzip the file and copy css/fonts/js to static folder under resources.
    • Import Bootstrap to index html page 
      <link rel="stylesheet" href="css/bootstrap.min.css">  
<link rel="stylesheet" href="css/bootstrap-theme.min.css">
<script src="js/bootstrap.min.js" type="application/javascript"></script>
